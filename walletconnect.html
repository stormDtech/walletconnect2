WalletConnect

WalletConnect is an Open protocol made to facilitate a secure connection between mobile cryptocurrency wallets and desktop application, such as Dapps. Transactions are made through an encrypted source protocol for connecting decentralized applications to mobile wallets with QR code scanning or deep linking. as a private key never leaves the user's device, their funds are never at risk and the chances of a hijack is very low. Users can interact securely with any Dapp from their mobile phone, making WalletConnect wallets a safer choice compared to desktop or browser extension wallets.

Fequently asked questions?

What is walletConnect?
WalletConnect helps you to use a Dapp with a mobile wallet. it works on both mobile and desktop and is becoming one of the main standard for Dapp access in wallets.

How can i use it?

  • Open the option to connect to a wallet.
  • Select a wallet among the wallet options you will be presented with
  • You will be prompted to import your wallet.

    • Is WalletConnect safe?
    Mobile application in general are not as secure as hardware wallet. Hardware wallet come with well-defined security and specific security solutions that allows them to more securely store and interact with users.
    However, walletconnect establishes a secure encrypted connection between your wallet and DApp. All informatiom transmitted between the wallet and DApp is encrypted symmetrically with a shared key that only these two parties posses. Whether you scan a QR code or use a deep link with a mobile wallet, the primary key isn't shared with the bridge server-that is, the key isn't linked to any thrid party, including the intermediary carrying out the secured transaction.
